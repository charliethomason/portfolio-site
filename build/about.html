About Charlie Thomason

My full name is Charles Lee Thomason, Jr. and I’m a Front-end Web Developer and Artist living in Chicago, Illinois. I was born in Red Bank, New Jersey in 1984, went to high school & college in Kentucky, and moved to Chicago in 2008.

My specialties as a Web Developer include web accessibility, responsive CSS, cross-browser compatibility, and rapid prototyping. I am currently employed full-time as a Presentation Layer Engineer for Razorfish, where I’ve been happily coding since January 2013. I studied web development at The Starter League (then known as Code Academy) in early 2012, and received my Masters in New Media Studies from DePaul University in 2010.

I've always been a visually creative person, and earned my Bachelor of Fine Arts in Art Studio from the University of Kentucky in 2007. Today, I still enjoy doing art as a hobby. I especially enjoy working with watercolor and ink painting, drawing, and photography. In 2012, I self-published a photography book called Open Range. Since 2014, I've been working on paintings of birds from the Chicagoland area.

I currently live in the South Loop neighborhood of Chicago with my fiancé, Amanda, and our 2 cats, Baxter and Chloe. I love movies, auto racing, LEGO, and playing guitar, bass, and piano. I am a happy, healthy, open-minded person and love traveling the world.

About this website

Although this website does not feature a true portfolio of my work as a Web Developer, I hope that the website itself will function as an example of my front-end coding skills. The entire site I designed and built from scratch. The only things on this site I did not personally create are jQuery and the imagesLoaded plugin (used for the "loading" effect in the art & photo galleries). The content is all static HTML, built using Handlebars and Grunt. The CSS was compiled from Sass, and borrows from Eric Meyer's CSS Reset. The galleries were custom-built with my own JavaScript. The site is hosted with Github Pages. The blog is hosted on Tumblr, using a custom, responsive Tumblr theme I built from scratch, and a customized version of Fancybox. If you want to see more of how the site is built, feel free to view the source code for this website. If you're interested in how the art and photo galleries work, you can view the JavaScript I wrote here.
